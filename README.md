# CSP451 CheckPoint 1

This project is part of Week 1 CheckPoint 1 for CSP451. It demonstrates basic Git workflows, repository setup, and documentation.

## Installation
1. Download or clone the repository.
2. Open the project folder on your computer.
3. Open `index.html` in your browser.

## Usage
- Open `index.html` in a browser to view the page.
- Open the browser console to see JavaScript output from `script.js`.
